# AWS Academy — Modules 1–4

- [Assignments](assignments.md)

This README summarizes **Modules 1 through 4** of the AWS Academy course. Access the actual content, labs, and knowledge checks in the [AWS Academy learning portal](https://awsacademy.instructure.com) after you receive your invite and log in with your @scranton.edu email.

## Requirements for Modules 1–4

To satisfy this part of the course you must:

1. **Pass the knowledge check** for each of Modules 1, 2, 3, and 4 with a **passing grade** (as defined in the AWS Academy course).
2. **Complete the lab** (activity) for **Module 1**, **Module 2**, **Module 3**, and **Module 4** in the AWS Academy portal.

Your instructor will verify completion via the AWS Academy course progress. If you are unsure what counts as passing or which lab to complete, check the course instructions in the portal or ask your instructor.

---

## Module 1: Cloud Concepts

| | Minutes |
|--|--------|
| **Lecture** | 45 |
| **Activity / Lab / Demo** | 5 |
| **Knowledge Check** | 10 |
| **Total** | **60** |

**What you’ll cover:** Introduction to cloud computing—what the cloud is, benefits (e.g. agility, elasticity, cost), and high-level service models (IaaS, PaaS, SaaS). Sets the foundation for the rest of the course.

---

## Module 2: Cloud Economics and Billing

| | Minutes |
|--|--------|
| **Lecture** | 45 |
| **Activity / Lab / Demo** | 45 |
| **Knowledge Check** | 10 |
| **Total** | **100** |

**What you’ll cover:** How AWS pricing and billing work—pay-as-you-go, cost allocation, pricing models, and the AWS billing and cost management tools. The lab time gives hands-on practice with cost-related features.

---

## Module 3: AWS Global Infrastructure Overview

| | Minutes |
|--|--------|
| **Lecture** | 25 |
| **Activity / Lab / Demo** | 35 |
| **Knowledge Check** | 10 |
| **Total** | **70** |

**What you’ll cover:** AWS Regions, Availability Zones, and edge locations. You’ll see how to choose regions and how global infrastructure supports reliability and low latency.
